FAQ
  • 무료 배송 및 반납
  • 제품 관련

카드를 등록하면 바로 결제가 이뤄지나요?

아니요. 한 달 무료 사용 기간이 종료되는 시점부터 결제가 시작됩니다.

선택한 결제 플랜이 지난 후엔 스위처 소유권이 제게 오나요?

아니요 스위처는 렌탈 방식입니다. 더 이상 사용을 원치 않으면 앱 속 '반납 신청'을 통해 스위처를 보내주셔야 해요.

사용 도중 반납할 경우 지불 금액은 어떻게 계산되나요?

선택한 결제 플랜에 따른 남은 기간에 대한 금액이 계산되어 환불 됩니다.
예1) '1달 사용' 선택 후 15일만 사용하고 반납을 하셨을 경우 결제한 1,800원 중 16일 치 금액을 환불받습니다.
(일 별 금액 58원 X 16일 = 928원).

반납은 어떻게 하나요?

스위처 앱 속에서 “반납 신청”을 진행하면 우체국 택배 기사님께서 전화 후 방문하여 제품을 수거해 가십니다.

지불 후 사용인 가요? 사용 후 지불인 가요?

지불 후 사용하는 방식입니다.

1개의 스마트폰으로 다수의 스위처 제어 가능한가요?

네 가능합니다. 1개의 스마트폰으로 다수의 스위처 제어 가능함은 물론 다수의 스마트폰으로 1개의 스위처 제어 가능합니다. 앞으론 매일 밤 불 꺼달라고 싸우지 마세요.

스위치 버튼이 2개 중 1개의 버튼만 사용하는데 1구 스위처를 신청해야 하나요? 2구 스위처를 사용해야 하나요?

1구 스위처 설치 시 스위처 뒷면 기어와 사용하는 버튼의 위치를 맞추면 사용 가능합니다. 하지만 1구 제품이 설치 되지 않을 수 있으니 구 스위처를 선택해주세요.

스위치 커버 위에 밸크로를 붙이면 흔적이 남지 않나요?

네 흔적이 남지 않습니다. 밸크로에 사용되는 접착제는 3M 제품으로 제거 시 아무런 흔적이 남지 않습니다.

배터리 충전시간 및 지속시간은 얼마나 되나요?

3시간 충전으로 약 50일간 사용 가능하십니다.
